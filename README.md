#### Features:

- Music
- Gif Search
- Translation
- Chat (wit.ai)

#### In Progress:

- Reminders

#### Installation:

1. Clone or download the repository
2. Install the dependencies:

```sh
$ cd koebot
$ npm install
```

3. Update the bot settings in config.json with your token and user id
4. Start the bot:

```sh
$ npm run start
```
