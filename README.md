#### Features:

- Music
- Gif Search
- Translation

#### Usage:

1. Clone or download the repository
2. Install the dependencies:

```sh
$ cd koebot
$ npm install
```

3. Update tokens.json with your token and user id
4. Start the bot:

```sh
$ npm run start
```
